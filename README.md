# Учебный проект Brain-Games в рамках обучения разработке Python на платформе Хекслет https://ru.hexlet.io/

# Включает в себя несколько консольных игр для разминки ума.
## Калькулятор (brain-calc)
  Суть игры в следующем: пользователю показывается случайное выражение, 
  например, 35 + 16, которое нужно вычислить и записать правильный ответ.
##  Проверка на четность (brain-even)
  Суть игры в следующем: пользователю показывается случайное число. 
  И ему нужно ответить yes, если число чётное, или no — если нечётное:
##  НОД - наибольший общий делитель (brain-gcd)
  Суть игры в следующем: пользователю показывается два случайных числа. 
  Пользователь должен вычислить и ввести наибольший общий делитель этих чисел.
##  Простое число (brain-prime)
  Суть игры в следующем: пользователю показывается число. 
  Пользователь должен определить простое число или нет.
##  Арифметическая прогрессия (brain-progression)
  Показываем игроку ряд чисел, который образует арифметическую прогрессию, 
  заменив любое из чисел двумя точками. Игрок должен определить это число.

# Установка:
    make install
    make build
    make package-install

# Запуск
    brain-calc
    brain-even
    brain-gcd
    brain-prime
    brain-progression
